# 🌍 Translations

AndroidAutoX is open to translations. Translations live in
`app/src/main/res/values-<locale>/strings.xml` — to add or fix a language, copy
the English source from `app/src/main/res/values/strings.xml` into the matching
`values-<locale>/strings.xml`, translate the string values, and open a pull
request.

A few notes:

- English (`app/src/main/res/values/strings.xml`) is the source of truth. Always
  base translations on it.
- The app name is a non-translatable proper noun (`AndroidAutoX`) and should not
  be localized.
- Please don't reupload the APK elsewhere; just link people to the download page.

Translators are credited in [`THANKS.md`](THANKS.md) — thank you! 🙏
